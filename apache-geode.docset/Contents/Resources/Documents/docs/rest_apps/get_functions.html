List all registered Geode functions in the cluster.

Resource URL

http://<hostname_or_http-service-bind-address>:<http-service-port>/gemfire-api/v1/functions

Parameters

None.

Example Request

GET /gemfire-api/v1/functions
Accept: application/json

Example Success Response

Response Payload: application/json

200 OK
Content-Length: <#-of-bytes>
Content-Type: application/json
Location: https://localhost:8080/gemfire-api/v1/functions

{
    "functions": [
        "AddFreeItemToOrders",
        "GetRegions",
        "GetDeliveredOrders"
    ]
}

Error Codes

Status Code Description
404 NOT FOUND Returned if no functions are found in the cluster.
500 INTERNAL SERVER ERROR Error encountered at Geode server. Check the HTTP response body for a stack trace of the exception.
